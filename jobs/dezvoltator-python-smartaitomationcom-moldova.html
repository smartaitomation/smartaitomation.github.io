Dezvoltator Python @ SmartAitomation.com Moldova

Post de munca vacant în rolul de dezvoltator Python la SmartAItomation Moldova

În rolul dezvoltatorului Python la SmartAitomation Moldova, veți crea și menține interfețe web pentru soluții care folosesc algoritmi de Inteligență Artificială pentru clienții noștri. Vei avea posibilitate de a participa la implementarea, antrenare, testarea și monitorizarea algoritmilor de Inteligență Artificială împreună cu experți în domeniu. Veți crea și testa algoritmi inteligenți pentru chatbot folosind api Telegram care vor avea aplicații în business, cultura, turism si educatie.

Cerinte:

  • Experiență în gestionarea mai multor medii virtuale Python (pip, conda)
  • Experiență cu un flux de lucru CI / CD automat și instrumente: depozite Git, Jenkins, pip / conda (optional)
  • Experiență practică în biblioteci, cum ar fi pandas, numpy, scipy, opencv
  • Competență în tehnologii de containere precum Docker
  • Înțelegerea de bază a sistemelor de orchestrare a containerului, de preferință Kubernetes
  • Cunostinte Flask sau Django
  • Experienta cu librariile tensorflow, keras, pytorch, pandas, matplotlib, numpy, opencv(optional)
  • Html/CSS/Javascript (opțional)
  • Abilități de lucru cu sitemul de operare Linux
  • Pasiune pentru un mediu de echipă
  • Abilități puternice de comunicare
  • Dorință și motivare mare de învățare

Postul este fulltime sau part-time, persoanele interesate pot trimite CV pe emailul: imosnoi@smartaitomation.com

De ce AI?

AI piermite proiectarea și dezvoltarea sistemelor informatice care pot îndeplini sarcini care necesită, de obicei, inteligența umană. Sarcini precum înțelegerea limbajului natural necesită abilități umane înnăscute, care sunt extrem de dificile pentru modelarea și rezolvarea calculatoarelor
